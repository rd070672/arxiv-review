# About

このサイトは、物性物理・計算材料科学・マテリアルズ・インフォマティクス・放射光実験・量子マテリアルに関する最近の論文を整理し、研究の流れを俯瞰するためのレビューサイトです。

日々公開される arXiv 論文を対象に、単なる新着紹介ではなく、物理的な本質、材料設計上の含意、計算や計測の位置づけが分かるように要点を再構成してまとめています。特に、電子・スピン・軌道・格子などの自由度の結合に着目し、量子効果がどのように巨視的な機能へつながるかという観点を重視しています。

## このサイトで扱う内容

- 物性物理に関する論文レビュー
- 計算材料科学に関する論文レビュー
- マテリアルズ・インフォマティクスに関する論文レビュー
- 放射光実験・量子ビーム計測に関する論文レビュー
- 量子マテリアルに関する論文レビュー

## レビュー方針

本サイトでは、以下の点を重視して論文を整理しています。

- どの物理現象・自由度・相互作用を扱っているか
- 何が新しく分かったのか
- その解釈を支える実験・理論・計算の根拠は何か
- 材料設計や機能開発の観点からどのような意味を持つか
- 今後の研究へどのような展開可能性があるか

## レビュー記事の作成方法

1. arXiv の新着論文を確認し、分野ごとに重要論文を選定します。  
2. Claude / Claude Cowork を活用して、論文レビューをマークダウン形式で作成します。  
3. 作成したレビュー記事をリポジトリに追加・整理します。  
4. GitHub Desktop で変更内容を確認し、commit と push を行います。  
5. GitHub Actions によりサイトが自動ビルドされ、VitePress + GitHub Pages で公開されます。

## 想定する読者

このサイトは、物性物理、材料工学、計算材料科学、量子ビーム計測、データ駆動型材料研究に関心をもつ学生・研究者を主な対象としています。専門分野の異なる読者にも流れが追いやすいよう、専門性を保ちながらも、論点が見える整理を心がけています。
