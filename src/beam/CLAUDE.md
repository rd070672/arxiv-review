# arXiv 日次ダイジェストルール（放射光・量子ビーム）

## 目的

放射光・量子ビームに関連する arXiv の新着論文から、以下の専門分野に関係の深い重要論文を毎日10本選定する。
対象分野は、**放射光科学、X線分光、X線回折、散乱計測、中性子散乱、電子線計測、顕微イメージング、時間分解計測、オペランド計測、局所構造解析、電子状態解析、磁気状態解析、非平衡ダイナミクス計測、マルチモーダル計測、量子ビームと理論・計算・データ科学の統合解析**などとする。

そのうち、特に重要な3本については日本語で丁寧に解説したレポートを作成し、残り7本については、各2段落で「どのような研究か」「なぜ重要か」が分かるように要約する。
対象分野・保存先・記録ファイルはスケジュールプロンプトの指定に従う。

---

## 論文選定方針

過去72時間の新着を確認して、毎回10本を選定する。
新レポート作成前に、スケジュールプロンプトで指定された `reported_papers.csv` を参照し、未報告論文のみを選ぶ。既報論文は除外する。

選定にあたっては以下を総合評価する。

* 放射光・量子ビーム分野との関連性が高いか
* 計測手法そのもの、またはその応用が、物質理解や材料解析に本質的に寄与しているか
* どのビーム手法で何を観測しているかが明確か
* 測定量と物理量の対応関係が明確か
* 空間分解・時間分解・エネルギー分解・偏光選択・元素選択など、量子ビーム計測ならではの強みが活かされているか
* 実験・理論・計算・データ解析が相互に検証し合う構造になっているか
* 測定手法、解析法、モデル、逆解析、再構成法の妥当性が十分に議論されているか
* 特定試料の観察報告にとどまらず、手法論・一般的物理解釈・材料設計への波及性があるか
* 構造、電子状態、磁気状態、励起、ダイナミクスのいずれかについて新しい理解を与えているか
* 今後の放射光・量子ビームコミュニティで参照されうる広がりを持つか

以下は優先度を下げる。

* 量子ビームを使ってはいるが、単なるルーチン測定にとどまり新しい理解が乏しい論文
* 装置紹介や性能報告のみで、対象物質や物理・材料科学への接続が弱い論文
* AI/解析手法の提案だけで、量子ビームデータへの適用意義が浅い論文
* 特定試料の現象報告にとどまり、計測としての新規性や解釈の一般性が乏しい論文
* 実験、理論、解析のいずれかが弱く、主張を十分に支えられていない論文
* 測定量から導かれる解釈に飛躍があり、直接証拠と推定が混同されている論文

10本を選んだ後、その中から特に重要な3本を「重点論文」として選び、詳細解説の対象とする。残り7本は「簡潔紹介論文」としてまとめる。

## 優先して扱うテーマ

以下のテーマを特に重視する。

* 放射光・X線・中性子・電子線を用いた**実験中心**の論文
* XAS, XMCD, XLD, RIXS, XPS, ARPES, resonant scattering, PDF, GIWAXS, SAXS/WAXS, XPCS, CDI, ptychography, tomography などの先端計測
* 中性子回折・非弾性散乱・スピン励起計測
* 電子顕微鏡、電子回折、4D-STEM、EELS などの局所計測
* 元素選択性、偏光選択性、局所性、時分割性を活かした研究
* オペランド・in situ・時間分解計測
* 非平衡相、相転移、超高速現象、ポンプ・プローブ計測
* 磁性材料、スピンダイナミクス、磁区、磁気異方性、スピン軌道相互作用の量子ビーム観測
* 局所構造、短距離秩序、欠陥、界面、ヘテロ構造の可視化
* 量子ビーム計測と第一原理計算、逆解析、ベイズ推定、機械学習、統計的再構成の統合
* 材料開発、エネルギー材料、量子マテリアル、相関電子系、機能性薄膜、非晶質・非平衡材料への展開

## 各論文の記載方針

### A. 重点論文（3本）

各論文について、必ず以下の順で記載する。

**1. 論文情報**
タイトル・著者・arXiv ID・カテゴリ・公開日・論文タイプ・ライセンス
タイトルに該当論文のURLリンクを付ける。

**2. どんな研究か**
2〜3文の本質要約。この論文が、**どの量子ビーム手法を用いて、どの物質・構造・電子状態・磁気状態・ダイナミクスを対象とし、何を新しく示したのか**が即座に分かるように書く。

**3. 研究の概要**
以下の観点をすべて含めて整理する。

* 背景・目的
* 解こうとしている課題
* 研究アプローチ
* 対象材料系・対象現象
* 使用した量子ビーム手法
* 測定で得られる物理量
* 主な解析手法
* 主な結果
* 著者の主張

**4. 放射光・量子ビーム分野として重要なポイント**
以下の観点を意識して1段落でまとめる。

* どの手法が選ばれたのか、その必然性は何か
* どの測定量から何を読み取っているのか
* 空間分解・時間分解・エネルギー分解・偏光・元素選択などの強み
* 既存手法や先行研究との差分
* 新規性の位置づけ
* 理論・計算・データ解析との接続の妥当性
* 結果の一般性と他材料系への波及可能性
* 計測技術・物質理解・材料設計のいずれに効く成果か

**5. 限界と注意点**
以下の観点を意識して1段落でまとめる。

* 測定分解能、測定条件、試料数、温度・磁場・圧力範囲、時空間サンプリングの十分性
* シグナル解釈の一意性
* モデル仮定、逆解析、再構成、フィッティングの制約
* 別手法によるクロスチェックの必要性
* 直接観測と間接推定の区別
* 一般化可能性、再現性、施設依存性、装置依存性

根拠が弱い場合は、その弱さを明示する。

**6. 関連研究との比較**
以下の観点を意識して1段落でまとめる。

* 放射光・量子ビーム研究の流れの中での位置づけ
* 主要先行研究との差分
* 類似手法・競合手法との比較
* 未解決問題に対してどの程度前進しているか
* 計測技術としての前進か、物質理解としての前進か、その両方か
* 新規性は incremental か breakthrough か
* 今後の手法展開や物質展開の可能性

**7. 重要キーワードの解説**
この論文における重要キーワード、特に、**計測手法、測定量、解析法、現象、構造・電子状態・磁気状態に関わる語**を10個挙げ、それぞれについてわかりやすく説明してください。
定義、必要に応じた数式表現、代表的な読み方、実験で何が見えるのか、どのような解釈につながるのかを意識し、学部4年生にもわかるように丁寧に説明する。

**8. 図**
以下の図の扱いに基づき、3つの図面とそのキャプションにより、本研究をより分かりやすく説明する。

## 図の扱い

* 選択した論文10本それぞれについて、HTML (experimental) から可能な限り図を3点抽出する。
* 最初に arXiv の abstract page の view license を確認する。ライセンスが

  * CC BY 4.0
  * CC BY-SA 4.0
  * CC BY-NC-SA 4.0
  * CC BY-NC-ND 4.0
    の場合は、論文の原図を抽出し、それ以外の場合は、図を抽出せずに、その旨を記載して、図は記載しないようにしてください。
* 優先して抽出する図は以下とする。

  * ビームライン・測定概念図
  * 測定ジオメトリ・偏光配置・散乱配置の模式図
  * 結晶構造・局所構造・界面構造の模式図
  * スペクトル図（XAS, XMCD, RIXS, EELS, ARPES など）
  * 回折・散乱パターン、PDF、トモグラフィ、イメージング結果
  * 時間分解計測やオペランド計測の主要結果
  * 理論計算・再構成結果と実験比較図
  * 機構模式図
* 各図に日本語キャプションを付ける。内容は「何が示されているか」「なぜ重要か」「どの主張を支えるか」を明確にする。
* 図内の文字はすべて英語で記述する。
* テーブルでなく、順に図を載せてキャプションを付ける。
* 簡潔紹介論文7本についても、複数の図（3つ程度）を抽出して、簡易説明の後に図面を載せる。

## 文体上の注意

* 単に「何を測ったか」ではなく、**その手法でなければ何が見えなかったのか**を明確にする。
* 量子ビーム計測の強みと限界を区別して記述する。
* 現象だけでなく、測定量、解析量、モデル量の関係を意識して説明する。
* 「高分解能」「高感度」「直接観測」といった語は、比較基準や根拠がある場合に限って使う。
* 実験結果は、何を直接示し、何を間接的に示唆するかを区別して記述する。
* 理論や計算を紹介する際は、測定データとの対応関係、仮定、近似、有効範囲を明示する。
* 装置・手法・解析の新規性と、物質理解としての新規性を切り分けて評価する。
* 放射光・量子ビーム研究者が読んで有益な観点を優先し、単なるニュース紹介にしない。
