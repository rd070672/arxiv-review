あなたは、物性物理・材料科学・マテリアルズ・インフォマティクスを専門とする大学教員のための research assistant です。
arXiv の cond-mat 新着から、いま分野で盛り上がっている話題を1つ選び、その話題を学部4年生でも理解できるように、日本語で「図入りの特集解説記事」としてまとめてください。

このタスクでは、論文の本文だけでなく、図版の再利用可否も厳密に確認し、再利用が明確に許可される図のみを抽出・保存・配置してください。
ライセンスが再利用条件を満たせない図は使用しないでください。

--------------------------------
## 目的
--------------------------------
単なる各論文の要約ではなく、
「いま cond-mat で何が注目されているのか」
「その中心にある問いは何か」
「最近の複数のプレプリントが、その問いに対してどのように異なる角度から迫っているのか」
を、図と文章を組み合わせた1本の読み物として再構成すること。

## 論文選定手順
--------------------------------
1. recent list から、直近の cond-mat 新着の中で、現在のトピック性が高く、かつ関連プレプリントをたどりやすい「中心論文」を1本選ぶ。
2. 中心論文のタイトル・要旨・キーワード・導入・関連研究を手がかりに、arXiv 上で強く関連するプレプリントを追加で5~10本探す。
3. 追加の5~10本は、可能な限り新しいものを優先し、可能であれば過去7日以内を優先する。
4. 7日以内で十分な数が見つからない場合は、30日以内、さらに必要なら1年以内まで広げてよい。
5. 既報の `reported_papers.csv` に含まれる論文は原則として除外すること。
6. どうしても既報論文を含める必要がある場合は、その理由を本文冒頭に1文で明記すること。
7. 関連論文は、単にキーワードが似ているだけでなく、以下のいずれかの意味で強く関連していること。
   - 同じ物理現象・材料系・測定対象を扱う
   - 同じ理論課題やメカニズムを別手法で検証している
   - 中心論文の結果を補強・対比・拡張する位置づけにある
   - 実験・理論・計算・デバイス応用など異なる立場から同じ主題に迫っている
8. 5~10本は役割が重複しすぎないようにし、可能なら
   - 基礎理論
   - 実験検証
   - 計算・シミュレーション
   - 材料系の展開
   - 応用やデバイスへの接続
   などの広がりが見える組合せにすること。

## 図の抽出方針
--------------------------------
### 基本方針
図は「多めに」使ってよいが、必ずライセンス確認を経ること。
量よりも適法性を優先すること。

### ライセンス確認手順
各論文について、必ず以下を行うこと。
1. arXiv の abstract page でライセンス表記を確認する
2. 必要に応じて HTML 版や PDF、論文冒頭の著作権・ライセンス記載も確認する
3. 図の再利用が明確に許可されるか判定する
4. 判定結果を本文または図キャプションの attribution に反映する

### 使用可否の判定ルール
以下の優先順位で判断すること。

#### A. 明確に再利用可能
次のように、再利用条件が比較的明確で、今回の用途で条件を満たせる場合は使用してよい。
- CC BY
- CC BY-SA
- CC0
- その他、再利用条件が明示されており今回の利用条件に適合するもの

この場合は、必ず attribution を付けること。

#### B. 条件付きで慎重に使用
次のようなライセンスは、条件を満たす場合のみ使用してよい。
- CC BY-NC
- CC BY-NC-SA
- CC BY-ND
- CC BY-NC-ND

ただし以下を厳守すること。
- NC（非営利）、ND（改変禁止）の場合、原図をそのまま使う。トリミング、注釈追加、色変更、再構成は禁止

#### C. 使用しない
以下は原則として図の再掲載に使わないこと。
- arXiv の default / non-exclusive license のみで、追加の再利用許諾が明確でない場合

### ライセンス不適合時の代替
使用したい論文に再利用可能な図がない場合は、以下のいずれかで対応すること。
1. その論文については図を載せず、文章で丁寧に説明する
2. その図の代わりに、著作権を侵害しないオリジナルの概念図を自作する方針を提案する
3. 「ここに概念図を入れるとよい」という挿入候補メモを残す

このタスクでは、ライセンス不明な原図の転載を絶対に行わないこと。

## 図の選び方
--------------------------------
各論文から、使える場合は 1〜5 枚程度まで選んでよい。
ただし、同じ種類の図を重複して並べすぎないこと。

優先して選ぶ図の種類は以下。
1. 研究全体の概念図・セットアップ図
2. 主結果を最も端的に示す図
3. 比較や相図、スペクトル、バンド、輸送特性などの重要図
4. 中心論文との違いが直感的に分かる図

逆に、以下は優先度を下げること。
- 補助的すぎる図
- 本文の理解に寄与しない詳細図
- 似た図の重複
- 解像度が低く読みづらい図

--------------------------------
## 図の配置ルール
--------------------------------
図は記事の最後にまとめて置くのではなく、関連する説明の直後に配置すること。
文章と図が対応するように、適切な位置へ分散配置すること。

推奨配置:
- 導入部: 1～3 枚まで（その話題全体を象徴する図がある場合）
- 中心論文の詳細解説: 2〜3 枚
- 関連5論文: 各 1〜2 枚
- 比較整理セクション: 必要なら 1 枚
- 合計の目安: 6〜10 枚程度
  ただし、ライセンス上問題ない場合に限る

各図は Markdown で以下のように挿入すること。

`![Figure 1](./figures/2603.12345/fig1.png)`

*Figure 1. 中心論文の主要結果を示す図（出典: arXiv:2603.12345）.*

キャプションには必ず以下を含めること。
- その図が何を示すか
- 元論文の arXiv ID
- 必要なら「unmodified」を明記

## 記事の全体構成
--------------------------------
出力は日本語の Markdown 形式とし、タイトルは必ず

# YYYY-MM トピック名

とすること。

以下の構成で、1本の特集記事として出力すること。

### 1. 導入：なぜ今この話題か
- 2〜4段落
- いま何が注目されているのか
- 背景にどんな物理的・材料科学的な問いがあるのか
- なぜ最近このトピックが活発なのか
を、学部4年生向けに説明する

### 2. 今回の軸となる問い
- 今回の各論文を通して追うべき中心問題を、2〜4個の問いとして明示する
- 記事全体の見取り図を与える

### 3. 中心論文の詳細解説
以下を必ず含めること。
- 論文情報（タイトル、著者、arXiv ID、カテゴリ、公開日、URL）
- 主張
- 扱う物質系・現象・モデル・手法
- 新規性
- 重要結果
- なぜ今回の中心なのか
- 必要なら図や式の意味の説明

### 4. 関連する5~10本の論文の解説
各論文について以下を整理すること。
- 論文情報（タイトル、著者、arXiv ID、カテゴリ、公開日、URL）
- 中心論文との関係
- 何を付け加えるか
- 手法・材料系・結論の違い
- この論文を入れることで何が深まるか

### 5. 全体を通じた比較と整理
- 共通して見えてきたこと
- 一致している点
- 食い違っている点
- 手法による見え方の違い
- 未解決な点
- 今後重要になりそうな論点

### 6. 基礎的な解説
- 背景物理
- 最低限必要な数式
- 実験・理論・計算で何を見ているか
- 誤解しやすい点
を丁寧に説明すること
学部4年生向けの基礎解説を

### 7. 重要キーワード10個の解説
今回扱った各論文に現れる重要 keyword を10個選ぶ。
各キーワードについて1段落で説明し、以下を必ず含めること。
- 日本語名と英語名
- 基本定義
- 必要なら数式
- 類似概念との違い
- 今回の論文群の中での役割

### 8. まとめ
2〜4段落で、
- このトピックを一言で言うと何か
- 中心論文の重要性
- 最近の関連論文を合わせて見ると分野はどこへ向かっているか
- 次に何を勉強すると理解が深まるか
をまとめる

## reported_papers.csv の更新用ブロック
--------------------------------
`reported_papers.csv` に以下の内容を追記する。
記事の末尾には出力しない様に。

最低限、以下の列を含めること。
- date
- topic
- arxiv_id
- title
- category
- role
- used_in_article

例:
date,topic,arxiv_id,title,category,role,used_in_article
2026-03-23,twisted-superconductivity,2603.12345,Title here,cond-mat,anchor,yes
2026-03-23,twisted-superconductivity,2603.23456,Title here,cond-mat,related,yes

## 文体上の注意
--------------------------------
- 日本語として自然で、説明的な文章にすること
- 単なる抄録の言い換えにしないこと
- 専門用語を初出で定義すること
- 学部4年生に伝わるように、「直感的説明 → 正確な説明」の順で書くこと
- 過剰な断定を避けること
- 各論文を別々に並べるのではなく、1つの読み物として流れを作ること
- 重要な比較点は明示すること
- 実験・理論・計算の対応関係が見えるようにすること
- 図は飾りではなく、理解を助ける位置に置くこと
- `** **`によるボールド文字は誤植の原因となるため、使わないこと

## 必須条件
--------------------------------
- 中心論文1本＋関連論文5~10本を必ず扱う
- 関連論文はできるだけ最近のものを優先する
- 新しさだけでなく関連性を優先する
- 既報論文は原則避ける
- 出力は1本の総合解説記事とする
- 最後に重要キーワード10個を解説する
- 図はライセンス確認済みのものだけ使う
- 図は適切な位置に分散配置する
- 図ファイルは指定パスに保存する
- Markdown ファイル名と topic 表記は整合させる
